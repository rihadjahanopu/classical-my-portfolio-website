# Rihad Jahan Opu | Full-Stack Developer Portfolio

![Rihad Jahan Opu Profile](public/assets/rihadprofile.jpg)

A premium, high-performance portfolio website built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. This project features advanced animations using **GSAP** and **Framer Motion**, providing a smooth and engaging user experience.

## 🚀 Live Demo
Check out the live site: [rihadjahanopu.com](https://www.rihadjahanopu.com)

## ✨ Key Features
- **Modern Tech Stack**: Built with Next.js 15 (App Router) and TypeScript.
- **Advanced Animations**: Smooth reveals and parallax effects powered by GSAP and ScrollTrigger.
- **Interactive UI**: Fluid motion and transitions using Framer Motion.
- **Smooth Scrolling**: Implemented Lenis for a premium scrolling feel.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **SEO Optimized**: Optimized metadata, open-graph tags, and semantic HTML for search engines.
- **Performance**: High Lighthouse scores with optimized image loading and efficient code.

## 🛠️ Tech Stack
- **Framework**: [Next.js 15](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) & [Framer Motion](https://www.framer.com/motion/)
- **Scrolling**: [Lenis](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide React](https://lucide.dev/) (where applicable)

## 📦 Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rihadjahanopu/classical-my-portfolio-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd classical-my-portfolio-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build
To create an optimized production build:
```bash
npm run build
```
